# Project agents

**Project-specific** subagents for the Objects of Agency website — committed to the
repo so they travel with the project and apply only here.

Put an agent here when its job is specific to *this* site. Reusable agents belong in
the **global** store at `~/.claude/agents/` — see `~/.claude/README.md` for the
global-vs-project rule.

Each agent is a markdown file with frontmatter (name, description, tools). Empty for now.
